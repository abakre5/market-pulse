# 🚀 H-1B Explorer

**Comprehensive H-1B Lottery Petition Analysis Tool**

A powerful web application for analyzing H-1B lottery petition data, providing deep insights into company hiring patterns, wage distributions, geographic trends, and policy impacts.

## 📊 Features

### 🏢 Company-Level Analysis
- **Detailed company insights** with hiring patterns and wage distributions
- **Wage level analysis** (I-IV) showing entry-level to senior positions
- **Top occupations** by salary and petition volume
- **Yearly trends** tracking hiring changes over time
- **Geographic distribution** across US states
- **Policy impact analysis** for H-1B rule changes

### 🤖 AI/ML vs Software Developers Comparison
- **Comprehensive analysis** of AI/ML Engineers vs Software Developers
- **Salary premium analysis** showing compensation differences
- **Growth trends** from 2020-2024
- **Wage level distribution** comparisons
- **Career progression insights** (entry vs senior positions)
- **Market share analysis** and industry trends

### 📈 Yearly Trends Analysis
- **Multi-year analysis** from 2020-2024
- **Policy impact assessment** for wage-based selection rules
- **Growth rate calculations** and trend visualization
- **Wage level evolution** over time
- **Geographic expansion** patterns

### 🗺️ State Explorer
- **Geographic analysis** by US states
- **Regional hiring patterns** and salary variations
- **State-by-state comparisons** for H-1B petitions
- **Local market insights** and opportunities

## 🛠️ Technology Stack

- **Frontend**: Streamlit (Python web framework)
- **Database**: DuckDB (in-memory SQL database)
- **Visualization**: Plotly (interactive charts)
- **Data Processing**: Pandas, NumPy
- **Styling**: Custom CSS with professional design

## 🚀 Quick Start

### Prerequisites
```bash
pip install -r requirements.txt
```

### Run the Application
```bash
streamlit run app.py
```

### Access the Application
Open your browser and navigate to `http://localhost:8501`
